PROJETO: GESTÃO DE GARANTIA – SISTEMA DE CONTROLE DE GARANTIA DE EQUIPAMENTOS

1 INTRODUÇÃO
A aquisição de equipamentos é uma prática comum no cotidiano, porém a falta de organização de documentos como notas fiscais e termos de garantia dificulta o acesso a essas informações quando necessário.
Diante desse problema, o projeto Gestão de Garantia propõe o desenvolvimento de um sistema mobile/web para armazenamento, controle e monitoramento de garantias de produtos, facilitando o acesso aos documentos e alertando sobre prazos de vencimento.

2 OBJETIVO
2.1 Objetivo Geral
Desenvolver um sistema para armazenamento e controle de garantias de equipamentos com alertas automáticos.
2.2 Objetivos Específicos
Armazenar documentos (nota fiscal e garantia)
Registrar dados da compra
Controlar prazos de garantia
Emitir alertas de vencimento
Suportar garantia estendida

3 JUSTIFICATIVA
A perda de documentos e a falta de controle de prazos podem gerar prejuízos financeiros. O sistema proposto facilita a organização e evita a perda de benefícios de garantia.

4 METODOLOGIA
O projeto será desenvolvido em ciclos quinzenais (sprints) ao longo de 3 meses.

5 FUNCIONALIDADES
Cadastro de equipamentos
Upload de documentos
Registro de dados da compra
Controle de prazo de garantia
Alertas automáticos
Cadastro de garantia estendida

6 TECNOLOGIAS
Frontend: React / React Native
Backend: Node.js
Banco: PostgreSQL
